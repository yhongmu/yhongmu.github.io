一、安装Node.js

​ Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，可以在非浏览器环境下，解释运行 JS 代码。参考地址：https://nodejs.org/en/

​ 下载完成后，打开cmd命令提示符，输入node -v、npm -v 出现版本号则安装成功。

二、安装Git

  • 安装教程：如何在windows下安装Git
  • 入门教程：Pro Git（中文版）
  • 基本操作：Git常用的基本命令

三、安装Hexo

​ Hexo 是一个快速、简洁且高效的博客框架，使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。 Hexo 的官方文档：https://hexo.io/zh-cn/

  • 打开Git Bash Here，输入安装hexo命令：npm install -g hexo-cli

  • 接下来，选择一个硬盘目录作为存放路径：hexo init D:\hexo

  • 然后进入该目录：cd e:\blog

  • 执行以下命令，系统会可以根据package.json文件中dependencies的配置安装所有依赖包：

    ​ npm install

  • 然后生成部署文件，启动本地服务:

​ hexo g # 或者hexo generate

​ hexo s # 或者hexo server，可以在http://localhost:4000/ 查看

  • Hexo 几个常用的命令：

​ hexo generate (hexo g) 生成静态文件，会在当前目录下生成一个新的叫做public的文件夹
​ hexo server (hexo s) 启动本地web服务，用于博客的预览
​ hexo deploy (hexo d) 部署博客到远端服务器
​ hexo new "postName" #新建文章
​ hexo new page "pageName" #新建页面

  • 可以简写成：

​ hexo n == hexo new
​ hexo g == hexo generate
​ hexo s == hexo server
​ hexo d == hexo deploy

  • 新建完成后，指定文件夹的目录如下：

  • 都是这些单词的首字母，更多的命令用法，可以参考Hexo官网

  • 现在我们打开http://localhost:4000/ 就可以看到我们刚才搭建的本地博客了，Hexo会默认生成一个Hello World的博文。
